# Interview Prep AI ML DS practice

10-day structured prep: Python/DSA, SQL, ML theory, Causal-ML, Bayesian methods, and marketing science.

## Structure

```
python/
  day1_arrays_hashing/          Hash maps, frequency counting, prefix products
  day2_sliding_window/          Two pointers, variable/fixed windows
  day3_trees/                   BFS/DFS, BST properties, recursion
  day4_graphs/                  BFS/DFS on grids, topological sort
  day5_intervals/               Sorting by start, heap-based merging
  day6_dynamic_programming/     1D/2D DP, memoization, tabulation
  day7_design_data_structures/  OOP design, heaps, ordered maps
  day8_ml_statistics/           A/B testing, metrics, uplift, Bayesian AB
  day9_ml_systems/              Recommenders, RAG, embeddings, BO
  day10_mock/                   Timed mock sessions

sql/
  day1_basics/                  Rolling averages, Top N per group
  day2_window_functions/        RANK, LAG/LEAD, cumulative sums, sessions
  day8_ab_testing_cohorts/      Experiment analysis, retention, funnels

causal_ml/                      Uplift modeling, DiD, PSM, geo experiments
bayesian_optimization/          BO concepts, price optimization
marketing_ml/                   MMM, audience segmentation, creative optimization
```

## Role-Specific Focus Areas
- **Causal-ML**: uplift modeling, difference-in-differences, PSM
- **Bayesian Optimization**: price optimization, creative selection
- **Marketing Science**: MMM, geo holdout experiments, audience modeling
- **Experimentation**: A/B at scale, CUPED, switchback tests
- **GenAI**: RAG pipelines, embeddings, vector search
